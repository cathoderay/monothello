1. Instalação:
==============
1.0 Requisitos
1.1 Windows
1.2 Linux


1.0 Requisitos:
===============

- Python 2.6.2

Se você utiliza o sistema operacional Windows, pule para a seção Windows.
O download do interpretador Python pode ser feito através da URL:
http://python.org/download

Escolha sua plataforma e baixe o instalador apropriado


1.1 Windows:
==============
O software acompanha um instalador do interpretador Python para Windows.

Se você não possuir o interpretador Python instalado, execute o instalador:
python-2.6.2.msi

Após a instalação, execute o arquivo monothello.py


1.2 Linux:
==============
Na maior parte das distribuições linux, Python já vem instalado. Você só precisa-
ria executar monothello.py

Se você possui uma distribuição debian-like e não possui python instalado, tente:
$ sudo apt-get install python

Pode ser preciso também instalar o módulo python-tk, portanto, execute também:
$ sudo apt-get install python-tk

Se não for possível prosseguir com sucesso, tente pelo site oficial:
http://python.org/download
